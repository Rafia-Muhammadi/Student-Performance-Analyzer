# Student-Performance-Analyzer

A Data analysis project using pandas, Matplotlib And Seaborn.

##Features
- Calculate Total Marks
- Calculate Average Marks
- Student Ranking
- Grade Calculation
- Subject-wise Analysis
- Visualization

## Technologies Used

- python
- Pandas
- Matplotlib
- Seaborn

  ## Visualization

  - Subject Average Bar Chart
  - Top 10 Students Chart
  - Grade Distribution
  - Correlation Heatmap
  - Box plot
  - Rank Vs Average
  - Pass Vs Fail
 
    ### Requirements

     Pandas
     Matplotlib
     Seaborn
